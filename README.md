## 작업기간

> 2022.12 ~ 2023.01 (1개월)

## 내용
> 제일 선호하는 쇼핑몰의 단점인 고정되어있지 않은 메인 메뉴와 추천 상품란을 추가하여 편리함과 높은 접근성을 추가했다. <br>
또한 미니멀한 느낌을 그대로 살려 제작했으며 가시성이 좋은 슬라이드를 추가하여 직관성을 높였다.
